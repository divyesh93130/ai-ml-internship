# 📊 Student Score Analyzer

This project was developed for **Assignment 01** of the **MINDS INVASION AI Internship**.

## 📌 About the Project

The Student Score Analyzer is a Python application that analyzes students' marks and generates useful statistics using **NumPy**. It also assigns grades based on predefined score ranges and provides an overview of class performance.

## ✨ Key Features

- Stores and processes student marks
- Assigns grades automatically:
  - **A** → 85 and above
  - **B** → 70 to 84
  - **C** → 50 to 69
  - **F** → Below 50
- Calculates:
  - Average (Mean)
  - Median
  - Highest Score
  - Lowest Score
  - Standard Deviation
- Counts students who scored above the class average
- Displays all **A-grade** students using NumPy Boolean Masking
- Supports adding extra student marks for updated analysis
- Identifies the grade category with the highest number of students
